# RTD-Analyzer-COS
Oprogramowanie dedykowane do cyfrowej obróbki i analizy danych eksperymentalnych uzyskanych z modelu wodnego urządzenia do ciągłego odlewania stali (COS). Aplikacja automatyzuje proces wyznaczania charakterystyk przepływu ciekłej stali w kadzi pośredniej na podstawie analizy krzywych rezydencji RTD typu F.
